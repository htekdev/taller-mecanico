# 🔧 Taller Mecánico - Sistema de Gestión

Sistema de gestión para taller mecánico con seguimiento de clientes, trabajos y finanzas.

## Características

- **Gestión de Clientes**: Registra clientes con nombre, teléfono y vehículo
- **Registro de Trabajos**: Rastrea cada servicio con mano de obra y costo de refacciones
- **Resumen Mensual**: Vista financiera con:
  - Total cobrado
  - Gastos en refacciones
  - Mano de obra total
  - Ganancia neta
  - Margen de ganancia

## Tecnología

- **Next.js 15** con App Router
- **TypeScript** para type safety
- **Tailwind CSS** para diseño responsive
- **localStorage** para persistencia de datos en el navegador

## Desarrollo Local

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Despliegue

Desplegado en Vercel. Los datos se guardan localmente en cada navegador.
